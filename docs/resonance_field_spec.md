## 2. Boundary Operators and Hodge Theory

### 2.1 Chain Groups

C_k(K) = free abelian group generated by oriented k-simplices.

### 2.2 Boundary Operator

∂_k: C_k → C_{k-1} defined by:

$$∂_k[v_{i_0}, ..., v_{i_k}] = \sum_{j=0}^{k} (-1)^j [v_{i_0}, ..., \hat{v}_{i_j}, ..., v_{i_k}]$$

Fundamental property: ∂_{k-1} ∘ ∂_k = 0 (verified in tests).

### 2.3 Hodge Laplacian

$$L_k = ∂_{k+1} ∂_{k+1}^T + ∂_k^T ∂_k = L_k^{up} + L_k^{down}$$

### 2.4 Hodge Decomposition

Any k-signal f ∈ C_k decomposes uniquely:

$$f = \underbrace{∂_k^T g}_{\text{gradient}} + \underbrace{∂_{k+1} h}_{\text{curl}} + \underbrace{φ}_{\text{harmonic}}$$

where φ ∈ ker(L_k). The harmonic component is the topological invariant — the "soul."

## 3. Resonance Dynamics

### 3.1 State Evolution

Module states x_i(t) ∈ ℝ^d evolve via iterative resonance:

$$x_i^{(n+1)} = \tanh\left(0.8 \cdot x_i^{(n)} + \sum_{j≠i} w_{ij} \cos(θ_j - θ_i) \cdot x_j^{(n)} \cdot 0.2\right)$$

### 3.2 Convergence Criterion

$$\max_i \|x_i^{(n+1)} - x_i^{(n)}\|_2 < ε$$

The tanh nonlinearity guarantees bounded states (Lyapunov stability).

## 4. Use-Dependent Plasticity (Hebbian)

### 4.1 Weight Update Rule

$$w_{ch}(t+1) = w_{ch}(t) + η \cdot a_{ch}(t) \cdot \text{trace}_{ch}(t) - λ \cdot w_{ch}(t)$$

where:
- η = learning rate (LTP)
- λ = decay rate (LTD / atrophy)
- a_{ch} = channel activation
- trace_{ch} = eligibility trace (exponential moving average of activations)

### 4.2 Eligibility Trace (STDP approximation)

$$\text{trace}_{ch}(t) = τ \cdot \text{trace}_{ch}(t-1) + a_{ch}(t), \quad τ = 0.95$$

### 4.3 Homeostatic Plasticity

Total synaptic budget is conserved:

$$\sum_{ch} w_{ch} = N_{channels} \quad \text{(enforced by rescaling)}$$

### 4.4 Neural Darwinism (Pruning)

Channels with w < threshold are candidates for pruning (Edelman 1987).

## 5. Kuramoto Synchronization

### 5.1 Pairwise Phase Dynamics (Classical)

$$\frac{dθ_i}{dt} = ω_i + K_1 \sum_{j=1}^{N} w_{ij} \sin(θ_j - θ_i)$$

### 5.2 3-Body Phase Coupling (Millán et al. 2020)

For each triangle σ = {i, j, k} containing vertex i:

$$\frac{dθ_i}{dt} += K_2 \sin(θ_j + θ_k - 2θ_i)$$

This creates explosive synchronization transitions and multistability.

### 5.3 Higher-Order Generalization (k-body)

For each k-simplex σ containing vertex i, with σ\i = {j₁,...,jₖ}:

$$\frac{dθ_i}{dt} += K_3 \sin\left(\sum_{m=1}^{k} θ_{j_m} - k \cdot θ_i\right)$$

### 5.4 Order Parameter

$$r = \left|\frac{1}{N} \sum_{j=1}^{N} e^{iθ_j}\right| \in [0, 1]$$

r → 1: full synchronization (coherent expression)
r → 0: desynchronization (internal processing)

## 6. Free Energy Principle

### 6.1 Variational Free Energy

$$F = E_q[\log q(s) - \log p(o,s)]$$

Approximated as precision-weighted prediction error:

$$F ≈ \frac{1}{2} π \sum_i (o_i - μ_i)^2$$

### 6.2 Belief Update (Gradient Descent on F)

$$μ_i(t+1) = μ_i(t) + α \cdot π \cdot (o_i - μ_i(t))$$

## 7. Emergence Metrics

### 7.1 Integrated Information (Φ)

$$Φ ≈ 1 - \frac{\max_i \text{Var}(x_i)}{\text{Var}(\text{flatten}(x))}$$

### 7.2 Criticality (χ)

## 8. Higher-Order Signal Propagation (pro/max)

### 8.1 Multi-Body Tensor Contraction

For a k-simplex σ = {v₀,...,vₖ} with target vₜ, sources S = σ\{vₜ}:

$$\text{contribution}_{vₜ} = γ \cdot w_σ \cdot \prod_{v \in S} \tanh(\bar{x}_v) \cdot \frac{1}{|S|}\sum_{v \in S} x_v$$

where γ is the higher-order gain (pro=0.15, max=0.25).

This implements AND-gate semantics: a channel fires strongly only when ALL source modules are co-active. Creates synergistic effects absent in pairwise coupling.

### 8.2 Tier Differentiation

- **lite**: Only pairwise propagation. Linear superposition of signals.
- **pro**: Adds 3-body and 4-body interactions. Nonlinear synergies emerge.
- **max**: Full 7-body global interaction. System exhibits true holistic behavior.

## 9. Dissipative Structure (Prigogine 1977)

### 9.1 Residual Decay

Between resonance cycles, states decay by factor ρ = 0.7:

$$x_i^{(0, \text{new cycle})} = ρ \cdot x_i^{(\text{final, prev cycle})}$$

### 9.2 Per-Iteration Dissipation

Within each resonance iteration, after tanh activation:

$$x_i^{(n)} \leftarrow x_i^{(n)} \cdot (1 - δ), \quad δ = 0.02$$

### 9.3 Bounded Steady State

The combination of tanh saturation + dissipation guarantees:
- Energy is bounded: E < N·d/2 (where N=modules, d=state_dim)
- System reaches steady-state oscillation (Kuramoto self-sustaining)
- No external input → gradual energy decay to attractor
