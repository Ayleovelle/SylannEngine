## 0. 记号、状态空间与适配器

立式，仿射双射适配：

```
φ(x) = (x+1)/2 ,   φ⁻¹(u) = 2u−1 .
```

维序固定为 `_DIM_NAMES = (warmth, arousal, valence, tension, curiosity, repair_pressure,
expression_drive, boundary_firmness)`。

**站姿假设（全文引用）**

## 1. 快通道 I：饱和更新的不变性

每回合语义更新（`saturating_update`）：

```
u⁺ᵢ = uᵢ + Gᵢ·aᵢ⁺·(1−uᵢ) − Gᵢ·aᵢ⁻·uᵢ ,    aᵢ⁺ = max(aᵢ,0), aᵢ⁻ = max(−aᵢ,0).
```

注意 aᵢ⁺·aᵢ⁻ = 0（至多一项非零）。

**定理 1（不变性与单调性）.** 设 (A2)(A3)。若 u ∈ [0,1]^8，则 u⁺ ∈ [0,1]^8。且
(i) aᵢ ≥ 0 ⟹ u⁺ᵢ ∈ [uᵢ, 1]；aᵢ ≤ 0 ⟹ u⁺ᵢ ∈ [0, uᵢ]（符号一致的单调移动）；
(ii) 增量在边界消失：|u⁺ᵢ − uᵢ| ≤ Gᵢaᵢ⁺(1−uᵢ) + Gᵢaᵢ⁻uᵢ → 0 当 uᵢ → 相应边界；
(iii)（红队修订）**边界可达性由 Gᵢaᵢ 定**：若某步 Gᵢ|aᵢ| = 1（(A2)(A3) 闭区间下合法，
如 G = 1、a = ±1），该步**一步精确到达**相应边界（u⁺ = 1 或 0）；若过程始终
Gᵢ|aᵢ| ≤ c < 1，则边界不可在有限步内由内部到达（每步至多走剩余距离的 c 倍，渐近逼近）。

*证明.* 情形 aᵢ ≥ 0：u⁺ᵢ = (1−Gᵢaᵢ)uᵢ + Gᵢaᵢ·1。由 (A2)(A3) 有 Gᵢaᵢ ∈ [0,1]，故 u⁺ᵢ 是
uᵢ 与 1 的凸组合，落在 [uᵢ,1] ⊆ [0,1]。情形 aᵢ < 0：u⁺ᵢ = (1−Gᵢaᵢ⁻)uᵢ，系数 ∈ [0,1]，故
u⁺ᵢ ∈ [0,uᵢ]。(ii) 由乘性因子 (1−uᵢ)、uᵢ 直接得到。(iii) Gᵢaᵢ = 1 时凸组合权重全落
边界端点（数值验证：`saturating_update([0.5]*8,[1]*8,[1]*8)` 恰返回 1.0）；
Gᵢ|aᵢ| ≤ c < 1 时按几何比例逼近。∎

**注 1.1（`validate_gain` 守的是什么——红队修订）.** 若允许 Gᵢaᵢ > 1，公式值
u⁺ᵢ = Gᵢ > 1 越界；但出厂代码的 `saturating_update` 内部有防御性 `_clamp01`，越界
misconfiguration 下**集合不变性照样成立**（G = 5 时静默返回 1.0，不逃逸）。故
`validate_gain` 的真实职责不是保 [0,1] 不变（那由内层 clamp 独立兜底），而是：
（a）**保本定理的定量结论**——单调步长、(ii) 的紧界；clamp 兜底下 G > 1 会把平滑饱和
静默变成一步贴边（另一类 bug，不是逃逸）；（b）**fail-fast 诊断**——在回落手写规则前
把 misconfiguration 暴露出来，而非静默吞掉。

**注 1.2（非仿射等变）.** 乘性因子 (1−u)、u 硬编码单位区间边界，故本算子**不**与仿射映射
交换：native 帧直接套用会把"压向地板 0"错译成"拉向中点"（负效价事件的语义翻转）。
反例已入测试（`test_saturating_update_is_not_affine_equivariant`）。接 native base 必须
整体折进折出。

## 2. 快通道 II：墙钟衰减的收缩性

回合间隔的惰性衰减（`decay`，半衰期形式）：

```
D(u₀; ū, h, Δt) = ū + (u₀ − ū)·β ,   β = 2^(−Δt/h) ∈ (0,1] ,  h > 0.
```

**定理 2（闭式解、收缩、无过冲）.** D 是线性 ODE u̇ = −k(u−ū)（k = ln2/h）在 Δt 上的精确流。
对 Δt > 0，**且设 u₀ ∈ [0,1]（红队修订：此前提不可省——见注 2.2）**：
(i) 收缩：|D(u) − D(v)| = β·|u − v|，β < 1，唯一不动点 ū；
(ii) 不变性：D(u₀) 是 u₀ 与 ū 的凸组合（权 β 与 1−β），故 [0,1] 不变；
(iii) 无过冲：D(u₀) ∈ [min(u₀,ū), max(u₀,ū)]——衰减永不越过均衡（无振铃）。

*证明.* 解 ODE 得指数流，2 的幂只是换底。(i) 由 |β(u−v)|。(ii)(iii) 由 D = βu₀ + (1−β)ū、
β ∈ (0,1)。∎

**注 2.1（浮点退化情形）.** IEEE-754 下极端 Δt/h（如 h = 1e−3s、Δt = 1e6s）使 β 下溢为
精确 0.0，此时 D 一步**精确**落在 ū 上——与"渐近逼近、永不到达"的理想化表述不符，但
(i)–(iii) 全部照常成立（退化凸组合）。仅当"渐近"措辞被逐字引用时需注意。

**注 2.2（裸凸组合，无内层 clamp——红队 fatal 的根因）.** 与 `saturating_update`（内层
`_clamp01`）和 `contagion_blend`（防御性 `_clamp01`）不同，`decay` 是**裸**凸组合公式，
对越界输入不自愈：u₀ = 1.5 时 D ≈ 1.49 仍越界，且逐步滞留界外。故 (ii) 的前提
u₀ ∈ [0,1] 必须在**调用边界**强制——见定理 4 的双闸。

**引理 2.1（仿射等变——代码捷径的合法性）.** 对任意可逆仿射 ψ(x) = cx + d：
ψ(D(u₀; ū)) = D(ψ(u₀); ψ(ū))。因 D 是系数和为 1 的仿射组合，而仿射映射与仿射组合交换。
故实现中"base 留 native 帧、仅把 Φ_eq 经 φ⁻¹ 折回"与"整体折进单位帧衰减再折出"**逐位相等**
（property test 已锚定，误差 < 1e−12）。饱和更新无此性质（注 1.2），两条捷径不可混用。

## 3. Scar 粘滞：时变速率下的一致指数稳定

半衰期受伤痕负载调制（`half_lives`）：

```
hᵢ(t) = h_base,ᵢ · gᵢ(T) · sᵢ(t) ,   sᵢ(t) = min(1 + σ·Lᵢ(t), S̄) ,  σ = 1, S̄ = 3,
```

其中 Lᵢ(t) ≥ 0 为该维加权伤痕密度（`scar_density`：RAW 1.0 / CLOSING 0.8 / SCARRED 0.5 /
FADED 0.3），随愈合阶段推进而衰减。

**g 的界（红队修订）**：g_tension = max(0.3, 1 + (percept − 0.5)·2)。A6 的代码闸
（`_trait` 夹 [0,1]）使 percept ≤ 1 ⟹ g ≤ 2.0（在 TraitMemory 实际供值域 [0.05,0.95] 下
可达上界为 1.9；2.0 是刻意取整的宽松界）。**此前 g 无上界强制**——越域 trait
（percept = 5.0）可得 g = 10、h 放大 5 倍、k̲ 相应变小，极限下 k̲ → 0 即定理 3 宣称
"永不发生"的冻结心境；红队发现后已在 `_trait` 落闸（测试锚
`test_affect_boundary_contract.TestTraitDomainEnforced`）。

**定理 3（一致指数稳定，anti-ratchet）.** 由封顶 sᵢ ≤ S̄ 与 g ≤ g_max（现为代码强制），
速率 k(t) = ln2/h(t) ≥ k̲ := ln2/(S̄·h_base,max·g_max) > 0。于是**在 ū 固定的前提下**，
对任意伤痕历史，|u(t) − ū| ≤ e^(−k̲·t)·|u₀ − ū|（时变线性 ODE 的积分因子精确解 +
k(t) ≥ k̲ 放缩）——收敛速度可被伤痕拖慢至多 S̄·g_max 倍，但永不冻结。

*证明.* u̇ = −k(t)(u−ū)，精确解 |u(t)−ū| = |u₀−ū|·exp(−∫₀ᵗ k(s)ds) ≤ |u₀−ū|e^(−k̲t)。∎

**注 3.1（封顶的必要性）.** 无封顶时 L → ∞ ⟹ h → ∞ ⟹ k → 0：受伤维度的情绪永久
凝固（吸收性心境），且 L 由不可逆伤痕累积单调可增——封顶是打断"越伤越僵、越僵越伤"
棘轮的唯一一道数学闸。S̄ = 3 是标定先验；定理只依赖 S̄ < ∞。

**注 3.2（ū 固定的显式声明——红队修订）.** 本定理把 ū = Φ_eq(T) 视为常量。慢通道反思会
**同时**改变 k(t)（经 gᵢ(T)）与 ū（经 Φ_eq(T)）——联动情形不属本定理，由注 5.2 的
（更弱的、逐维定量的）跟踪界覆盖，勿把本定理引为联动情形的依据。另：本定理与 G 无关
（`half_lives` 不引用增益）——引理 6 的适用清单以此为准。

## 4. 全回合组合不变性

一个回合内可能依序发生：衰减（§2）→ 创伤注入（native 帧 tanh 演化，值域 (−1,1)）→
饱和更新（§1）→ 记忆传染（§8）→ 标签解析（只读）。

**定理 4（组合不变性——红队 fatal 后重述）.** 设回合开始时状态在不变集
K = [0,1]^8（native 帧 [−1,1]^8）内，则上述任意算子任意次序的有限复合把 K 映入自身；
特别地，有界性与事件次序无关。**"状态在 K 内"这一前提由两道代码闸在边界强制**：
（闸一）`ScarredState.from_dict` 复原时把 base 夹回 [−1,1]（损坏/手改快照的唯一入口；
合法快照恒等）；（闸二）`_affect_decay` 入口把待衰减向量夹回 [−1,1]（对未来其它写入者
的皮带）。

*证明.* 衰减与传染是凸组合（定理 2 + 注 2.2 的前提由闸强制、§8 且 `contagion_blend`
另有防御 clamp），饱和更新由定理 1（内层 `_clamp01` 双保险），tanh 演化值域
(−1,1) ⊂ φ⁻¹(K)。闭集对连续映射复合封闭。∎

**修订记录（红队 composition 镜头，裁定 BROKEN→修复）**：v1 的证明句"适配器对复原快照的
越界值有末端 clamp"为**假**——takeover 捷径（引理 2.1）恰恰绕过适配器，`from_dict` 当时
对 base 零校验，红队以 base = 1.5 数值复现了穿透（decay 后 1.4907，永久越界）。两道闸
即为此修复；攻击链全程入测试
（`test_affect_boundary_contract.TestRestoreClampsBase.test_corrupt_then_decay_stays_bounded`）。

**注 4.1（次序影响值，不影响界）.** "衰减必须在事件演化之前"（settle-then-evolve，
设计 §9）是**行为**要求（长静默后事件响应不被擦除），不是有界性要求——前者由
`test_event_survives_long_silence` 在真实 `step()` 上锚定，与本定理正交。

## 5. 慢通道：锚回弹的有界漂移半径

反思事件（墙钟冷却 Δ_c = 1800s 分隔）触发人格宏漂移。**出厂递推（红队修订——v1 写的是
另一个未出厂的系统）**：`slow_channel.py` 计算 `q · drift_step(...)`，q 同时调制推动项
**与**回弹项（A8）：

```
T_{n+1} = T_n + q_n·( η·û_n − ρ·(T_n − A) ) ,   q_n ∈ [0,1], ‖û_n‖∞ ≤ 1, 0 < ρ < 1,
```

即 e_{n+1} = (1 − q_n·ρ)·e_n + q_n·η·û_n（e_n = T_n − A）。A 为不可变锚（初值），
η 漂移步长，ρ 回弹强度。

**定理 5（不变球与不增性——按出厂系统重证）.** 令 B = {e : ‖e‖∞ ≤ η/ρ}。则：
(i) **不变性**：e_n ∈ B ⟹ e_{n+1} ∈ B，对**任意** q_n ∈ [0,1]；
(ii) **界外不增**：‖e_n‖∞ > η/ρ ⟹ ‖e_{n+1}‖∞ ≤ ‖e_n‖∞（q_n > 0 时严格减）；
(iii) 故 sup_n ‖T_n − A‖∞ ≤ max(‖T₀ − A‖∞, η/ρ)；从 T₀ = A 出发永远
‖T_n − A‖∞ ≤ η/ρ。
(iv) **收敛入球的条件**：进入 B 需 Σ q_n = ∞（如 q_n ≥ q̲ > 0 无穷多次）；若质量门
长期 q_n ≈ 0，e_n 停在当前值——**无回弹**。

*证明.* 映射 e ↦ e + q(ηû − ρe) 对 q 仿射，端点 q = 0（恒等，保 B）与 q = 1
（‖(1−ρ)e + ηû‖∞ ≤ (1−ρ)η/ρ + η = η/ρ，保 B）都落在凸集 B 内，两点连线段亦然，得 (i)。
(ii)：‖e_{n+1}‖∞ ≤ (1−q_nρ)‖e_n‖∞ + q_nη < ‖e_n‖∞ 当 ‖e_n‖∞ > η/ρ、q_n > 0。
(iii)(iv) 由 (i)(ii) 与 (1−q̲ρ) 的几何压缩。∎

**注 5.1（真正的约束层级——红队定量修订）.** 出厂先验 η = 0.30、ρ = 0.20、Q̄ = 1
（**结构界**，q_dc 的 clamp01 不可标定放宽）⟹ 球半径 η/ρ = 1.5 > 特质域半宽 0.45
（clamp [0.05,0.95]）——**本定理的球在当前先验下不束缚**。实际约束由紧到松：
（1）**共享 L1 步长预算** `_TICK_DRIFT_CAP = 0.05`/次反思（对五特质总和限幅；
从锚到 clamp 边界至少需 ⌈0.45/0.05⌉ = 9 次反思 × 1800s 冷却 ≈ 4.5h+ 持续同向极端事件）；
（2）clamp [0.05,0.95]（半径 0.45）；（3）本定理的球（1.5）。要让锚球主导需重标
η/ρ ≤ 0.45——**唯一可动的杠杆是 η/ρ**，Q̄ 不是。共享预算同时使漂移**维间耦合**（§0 的
独立性除外条款）。

**注 5.2（"可回弹"的诚实条件 + 双时标逐维定量——红队修订）.**
（a）**回弹是有条件的**：q 同时门控回弹项 ⟹ 长期低质量对话（q ≈ 0）下特质停在漂移处、
不弹回锚——"生活能弯折她、但改写不了她"的后半句成立（有界），前半句的"弯折后回弹"
**以持续有质量的对话为前提**（定理 5(iv)）。这是出厂语义（垃圾会话既不许推走也不许拉回
人格），不是缺陷，但必须说破。
（b）**双时标跟踪，逐维定量**：Φ_eq 对 T 的敏感度按 T-only 系数行和计算（R 是独立实参且
当前恒 0.5，不入 ΔT 扰动——v1 曾把 R 系数误加进行和）：warmth 0.20、arousal 0.35、
**valence 0.40（最大行）**、tension 0.25、curiosity 0.25、repair 0.15、expr 0.30、
boundary 0.30 ⟹ ‖Δū‖∞ ≤ 0.40·‖ΔT‖∞。跳变后一个冷却期 Δ_c 内闭合的缺口比例为
1 − 2^(−Δ_c/hᵢ)，**逐维差异巨大**：arousal（h = 30min = Δ_c）闭合 50%、expr ≈ 57%，

## 6. Delta-rule 增益可塑性（设计数学，未实现——A2 轨的前置）

拟议更新（每回合，表达结果反馈 quality_t 到达后）：

```
G_{t+1} = Π_{[ε,1]}( G_t + α·δ_t·φ_t ) ,
δ_t = clip(quality_t − q̂_t, [−1,1]) ,     q̂_t：quality 的 EMA（预测基线）,
φ_t ∈ [0,1]^8：per-dim 资格迹（近期 |a| 活动的泄漏累积）,
```

Π 为逐维投影到 [ε,1]（ε > 0 下限防增益死亡）。

**引理 6（投影不变性——红队降格自"分离定理"）.** 定理 1 对 G 的全部依赖仅为 (A3) 之界
（定理 4 经由定理 1 间接依赖；定理 2、3 与 G 无关——注 3.2）。故对**任意**序列
{G_t} ⊂ [ε,1]^8，定理 1–4 逐字成立：只要更新经投影 Π，增益学习不能**经由 G 的取值**
破坏 E 的有界性与收缩性。

*证明.* 逐条检查定理 1–4 证明对 G 的引用：仅 (A3)。Π 逐步强制该界。∎

**降格说明（红队 positioning 镜头）**：v1 称此为"分离定理"并列为具名贡献 C3——撤回。
经典分离原理（LQG）是需要实证的非平凡结果；本引理是对既有证明的**引用审计**（proofs
only use G's range ⟹ range-preserving updates preserve proofs），构造上即真。它**不**排除
时标耦合（学习率 vs 收缩率 vs 反思漂移的倒挂——注 6.1 的标定约束正是一个真·分离定理
需要证明而本文只能假设的东西），也不解除定理 4 的入域前提（现由代码闸强制）。它的价值
是工程契约：A2 实现只需测试 Π 的正确性 + 时标 conformance，无需重证系统安全——
一个有用的引理，不是一个定理级贡献。

**注 6.1（时标排序，量纲修订）.** v1 的排序把"每反思一次的 G 跳变"与"每回合步长 α"
直接比较——量纲错位（红队修正）。正确的全速率式（对回合间隔 Δt_turn 归一）：

```
k̲ ≫ α/Δt_turn ≫ (ηQ̄/ρ · |∂G/∂T|)/Δ_c
```

等价地乘 Δt_turn：k̲·Δt_turn ≫ α ≫ ΔG_reflect·(Δt_turn/Δ_c)——反思引发的 G 跳变须
**摊销到两次反思之间的回合数**再与 α 比较。另注意 Δt_turn 不是常量（墙钟间隔跨多个
数量级），故此约束是 regime 约束（常规对话节奏下成立），不是全域恒等式——入 conformance
时按 Δt_turn 分布的分位数检验。

**注 6.2（资格迹的必要性）.** 若 φ 恒 1，quality 的全局标量误差会同步推动全部 8 维增益
（credit assignment 失败，学到的只是全局兴奋度）。φ = 近期该维 |a| 活动 ⟹ 只有参与了
本轮情绪反应的维度领赏罚。这是把"学习"从噪声放大器变成信用分配器的结构件。

|u₀ − ū|（最大 ~0.85），(A2)(A3) 闭区间 ⟹ 一次全力事件位移可达 1——s̄ 的真上确界为

无条件成立的只有：换档需要真实跨带（不会由浮点噪声触发）。

**注 7.1.** 单次大事件（|a| 大）仍可一步换标——这是特性不是缺陷：真实事件应当立即
反映；死区只滤边界徘徊。最坏情形披露里的"1 回合反转"正是两个真实极端事件背靠背的

## 9. 定位与具名贡献（红队 positioning 镜头后重写）

**对标（对话代理架构）**：WASABI（Becker-Asano 2008，PAD 空间心境向人格基线指数衰减）、
ALMA（Gebhard 2005，人格推导的心境均衡 + 低通）、EMA/OCC/FAtiMA（appraisal 谱系）、
Park et al. 2023（poignancy/reflection，作用于记忆重要度与自由文本反思，非数值特质 ODE）。

**对标（心理学 affect dynamics——红队指出的缺失，此处补齐）**：本文标题借用的
"情感动力学"是既有心理学领域之名：Kuppens, Oravecz & Tuerlinckx 2010（DynAffect，
OU 过程心境模型，吸引子强度/惯性为**个体差异参数**）、Kuppens & Verduyn 2017（emotion
dynamics 综述）、Koval 等（情绪惯性与抑郁）、Post 1992（kindling：反复发作提高敏感性、
缩短发作间隔）；本仓库 `theoretical_grounding.md` 已为热池引用 Loossens et al. 2020
（Affective Ising Model）。**与 C1 的关系**：这些模型的惯性/吸引子强度是个体层（或情境层）
参数；C1 把时间常数绑到**会话内、逐事件、逐维、封顶且自愈的伤痕账本**上——机制化方向
与 kindling 的敏感化（更快发作）不同，更接近"受伤维度恢复变慢"（情绪惯性升高与抑郁
相关的方向）。**本文对该领域不做新颖性主张**，只主张下述 C1 在**已调研的对话代理架构**
中无对应物。

**具名贡献（收窄后的两件 + 一件工程契约）：**

- **C1 scar 耦合时间常数调制**：情绪衰减半衰期由不可逆伤痕账本调制、封顶、自愈

  `v26-super-standard-campaign.md`），在该范围内无对应物；与心理学 affect-dynamics
  文献的关系如上——是对"惯性可变"现象的一种**运行时安全有界的机制化**，非领域新颖性。
- **C2 锚回弹有界人格可塑性（工程契约级）**：定理 5 给出出厂递推的不变球与不增性；

- **（原 C3，降格）投影不变性引理**：见引理 6 及其降格说明——工程契约，不列为贡献。

**命名撞车处置**：poignancy/reflection 沿用 Park 词汇但语义不同（数值特质 ODE vs
记忆反思树）；标准文本中首次出现处加差异化脚注，中文名采"沉淀量/回望"以避免

poignancy——poignancy 只经 `reflection_ready` 决定反思**时机**，不进 T 更新的幅度项；
