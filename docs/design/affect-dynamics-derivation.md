## 0. 记号、状态空间与适配器

情感核 E 存储于 native 帧 `x ∈ [−1,1]^8`（`ScarredState.base`，tanh 有界）。动力学在单位帧
`u ∈ [0,1]^8` 立式，仿射双射适配：

```
φ(x) = (x+1)/2 ,   φ⁻¹(u) = 2u−1 .
```

维序固定为 `_DIM_NAMES = (warmth, arousal, valence, tension, curiosity, repair_pressure,
expression_drive, boundary_firmness)`。全文按维独立分析（所有算子逐维作用），故不失一般性
在标量上证明，向量结论逐维成立。

**站姿假设（全文引用）**

- (A1) 事件到达于离散回合；回合间墙钟间隔 Δt ≥ 0。
- (A2) appraisal 增量 a ∈ [−1,1]^8（`project_appraisal` 末端 clip 保证）。
- (A3) 增益 G ∈ (0,1]^8（`validate_gain` fail-closed 保证；越界即拒绝更新）。
- (A4) 均衡 Φ_eq ∈ [0.15,0.85]^8（`equilibrium` 末端 clamp；内收避免吸收态）。
- (A5) 传染强度 κ ∈ [0,1]、记忆指纹 m ∈ [0,1]^8。
- (A6) 人格特质 T 有 clamp [0.05,0.95]，锚 A（anchor）不可变。
- (A7) 本分析仅覆盖 E 律 regime；PEL 激活时 E 律夺权失效（config 互斥 + 代码 inert），
  PEL regime 的分析见 `v25-pel-core-techspec.md`，不在本文范围。

## 1. 快通道 I：饱和更新的不变性

每回合语义更新（`saturating_update`）：

```
u⁺ᵢ = uᵢ + Gᵢ·aᵢ⁺·(1−uᵢ) − Gᵢ·aᵢ⁻·uᵢ ,    aᵢ⁺ = max(aᵢ,0), aᵢ⁻ = max(−aᵢ,0).
```

注意 aᵢ⁺·aᵢ⁻ = 0（至多一项非零）。

**定理 1（不变性与单调性）.** 设 (A2)(A3)。若 u ∈ [0,1]^8，则 u⁺ ∈ [0,1]^8。且
(i) aᵢ ≥ 0 ⟹ u⁺ᵢ ∈ [uᵢ, 1]；aᵢ ≤ 0 ⟹ u⁺ᵢ ∈ [0, uᵢ]（符号一致的单调移动）；
(ii) 增量在边界消失：|u⁺ᵢ − uᵢ| ≤ Gᵢaᵢ⁺(1−uᵢ) + Gᵢaᵢ⁻uᵢ → 0 当 uᵢ → 相应边界；
(iii) 边界 0、1 不可在有限步内由内部到达（渐近逼近）。

*证明.* 情形 aᵢ ≥ 0：u⁺ᵢ = (1−Gᵢaᵢ)uᵢ + Gᵢaᵢ·1。由 (A2)(A3) 有 Gᵢaᵢ ∈ [0,1]，故 u⁺ᵢ 是
uᵢ 与 1 的凸组合，落在 [uᵢ,1] ⊆ [0,1]。情形 aᵢ < 0：u⁺ᵢ = (1−Gᵢaᵢ⁻)uᵢ，系数 ∈ [0,1)，故
u⁺ᵢ ∈ [0,uᵢ]。(ii)(iii) 由乘性因子 (1−uᵢ)、uᵢ 直接得到：从内部出发每步至多走剩余距离的
Gᵢaᵢ 倍。∎

**注 1.1（G ≤ 1 的必要性）.** 若允许 Gᵢaᵢ > 1，取 uᵢ = 0、aᵢ = 1 得 u⁺ᵢ = Gᵢ > 1，逃逸。
故 (A3) 的上界是紧的——这就是 `validate_gain` 为什么是硬闸而非风格偏好。

**注 1.2（非仿射等变）.** 乘性因子 (1−u)、u 硬编码单位区间边界，故本算子**不**与仿射映射
交换：native 帧直接套用会把"压向地板 0"错译成"拉向中点"（负效价事件的语义翻转）。
反例已入测试（`test_saturating_update_is_not_affine_equivariant`）。接 native base 必须
整体折进折出。

## 2. 快通道 II：墙钟衰减的收缩性

回合间隔的惰性衰减（`decay`，半衰期形式）：

```
D(u₀; ū, h, Δt) = ū + (u₀ − ū)·β ,   β = 2^(−Δt/h) ∈ (0,1] ,  h > 0.
```

**定理 2（闭式解、收缩、无过冲）.** D 是线性 ODE u̇ = −k(u−ū)（k = ln2/h）在 Δt 上的精确流。
对 Δt > 0：
(i) 收缩：|D(u) − D(v)| = β·|u − v|，β < 1，唯一不动点 ū；
(ii) 不变性：D(u₀) 是 u₀ 与 ū 的凸组合（权 β 与 1−β），故 [0,1] 不变；
(iii) 无过冲：D(u₀) ∈ [min(u₀,ū), max(u₀,ū)]——衰减永不越过均衡（无振铃）。

*证明.* 解 ODE 得指数流，2 的幂只是换底。(i) 由 |β(u−v)|。(ii)(iii) 由 D = βu₀ + (1−β)ū、
β ∈ (0,1)。∎

**引理 2.1（仿射等变——代码捷径的合法性）.** 对任意可逆仿射 ψ(x) = cx + d：
ψ(D(u₀; ū)) = D(ψ(u₀); ψ(ū))。因 D 是系数和为 1 的仿射组合，而仿射映射与仿射组合交换。
故实现中"base 留 native 帧、仅把 Φ_eq 经 φ⁻¹ 折回"与"整体折进单位帧衰减再折出"**逐位相等**
（property test 已锚定，误差 < 1e−12）。饱和更新无此性质（注 1.2），两条捷径不可混用。

## 3. Scar 粘滞：时变速率下的一致指数稳定

半衰期受伤痕负载调制（`half_lives`）：

```
hᵢ(t) = h_base,ᵢ · gᵢ(T) · sᵢ(t) ,   sᵢ(t) = min(1 + σ·Lᵢ(t), S̄) ,  σ = 1, S̄ = 3,
```

其中 Lᵢ(t) ≥ 0 为该维加权伤痕密度（`scar_density`：RAW 1.0 / CLOSING 0.8 / SCARRED 0.5 /
FADED 0.3），随愈合阶段推进而衰减；gᵢ(T) 有界（代码中 g ∈ [0.3, 2.0]）。

**定理 3（一致指数稳定，anti-ratchet）.** 由封顶 sᵢ ≤ S̄ 与 g 有界，速率
k(t) = ln2/h(t) ≥ k̲ := ln2/(S̄·h_base,max·g_max) > 0。于是对任意伤痕历史，
|u(t) − ū| ≤ e^(−k̲·t)·|u₀ − ū|（比较原理）——收敛速度可被伤痕拖慢至多 S̄ 倍，
但**永不冻结**。

*证明.* u̇ = −k(t)(u−ū)，k(t) ≥ k̲ ⟹ d|u−ū|/dt ≤ −k̲|u−ū|，Grönwall。∎

**注 3.1（封顶的必要性）.** 无封顶时 L → ∞ ⟹ h → ∞ ⟹ k → 0：受伤维度的情绪永久
凝固（吸收性心境），且 L 由不可逆伤痕累积单调可增——封顶是打断"越伤越僵、越僵越伤"
棘轮的唯一一道数学闸。S̄ = 3 是标定先验；定理只依赖 S̄ < ∞。

## 4. 全回合组合不变性

一个回合内可能依序发生：衰减（§2）→ 创伤注入（native 帧 tanh 演化，值域 (−1,1)）→
饱和更新（§1）→ 记忆传染（§8）→ 标签解析（只读）。

**定理 4（组合不变性）.** 上述任意算子任意次序的有限复合把不变集 K = [0,1]^8（等价地
native 帧 [−1,1]^8）映入自身。特别地，**有界性与事件次序无关**。

*证明.* 衰减与传染是凸组合（定理 2、§8），饱和更新由定理 1，tanh 演化值域 (−1,1) ⊂
φ⁻¹(K)，适配器对复原快照的越界值有末端 clamp。闭集对连续映射复合封闭。∎

**注 4.1（次序影响值，不影响界）.** "衰减必须在事件演化之前"（settle-then-evolve，
设计 §9）是**行为**要求（长静默后事件响应不被擦除），不是有界性要求——前者由
`test_event_survives_long_silence` 在真实 `step()` 上锚定，与本定理正交。

## 5. 慢通道：锚回弹的有界漂移半径

反思事件（墙钟冷却 Δ_c 分隔）触发人格宏漂移（`slow_channel` + `TraitMemory.anchor`）：

```
T_{n+1} = T_n + η·q_n·û_n − ρ·(T_n − A) ,   |q_n| ≤ Q̄, ‖û_n‖∞ ≤ 1, 0 < ρ < 1,
```

A 为不可变锚（初值），η 漂移步长，ρ 回弹强度。

**定理 5（漂移半径）.** 令 e_n = T_n − A。则 ‖e_{n+1}‖∞ ≤ (1−ρ)‖e_n‖∞ + ηQ̄，故

```
limsup_n ‖T_n − A‖∞ ≤ ηQ̄/ρ ,
```

且从 T₀ = A 出发任意有限 n 也满足 ‖e_n‖∞ ≤ (ηQ̄/ρ)(1−(1−ρ)ⁿ) < ηQ̄/ρ。
**无论 poignancy 历史如何**，人格永远停留在以锚为心、半径 ηQ̄/ρ 的盒内。

*证明.* 递推放缩 + 几何级数。∎

**注 5.1（"生活能弯折她、但改写不了她"）.** 这是慢通道的安全契约：漂移影响是有界的、
可回弹的；配合既有 clamp [0.05,0.95] 与逐次反思步长 cap，实际约束取三者最紧。代码先验
η = 0.30、ρ = 0.20 ⟹ 半径 ≤ 1.5·Q̄——即**约束由 clamp 与步长 cap 主导**，ρ 的回弹在
长时程上提供均值回归而非硬界。若产品要求锚盒主导，需 η/ρ ≤ 期望半径（标定项）。

**注 5.2（双时标跟踪，非正式）.** 快态以速率 ≥ k̲（定理 3）收缩到 Φ_eq(T)；T 仅在反思时刻
移动，每次 ‖ΔT‖∞ ≤ ηQ̄ + ρ‖e‖∞。Φ_eq 对 T 是分段线性 + clip（1-Lipschitz），系数行和 ≤ 0.5，
故均衡跳变 ‖Δū‖∞ ≤ 0.5‖ΔT‖∞。跳变后跟踪误差按 2^(−t/h) 衰减；冷却 Δ_c ≫ h ⟹ 反思间隔内
快态基本贴合新均衡。此处只主张**跟踪误差有界**（≤ 均衡跳变幅度 + 指数尾），不援引完整
奇异摄动/平均化定理——诚实边界。

## 6. Delta-rule 增益可塑性（设计数学，未实现——A2 轨的前置）

拟议更新（每回合，表达结果反馈 quality_t 到达后）：

```
G_{t+1} = Π_{[ε,1]}( G_t + α·δ_t·φ_t ) ,
δ_t = clip(quality_t − q̂_t, [−1,1]) ,     q̂_t：quality 的 EMA（预测基线）,
φ_t ∈ [0,1]^8：per-dim 资格迹（近期 |a| 活动的泄漏累积）,
```

Π 为逐维投影到 [ε,1]（ε > 0 下限防增益死亡）。

**定理 6（分离定理——安全与学习解耦）.** 定理 1–4 的证明只使用 G ∈ (0,1]，对**任意**
序列 {G_t} ⊂ [ε,1]^8 逐字成立。故：只要更新经投影 Π，**无论学习信号多错、多噪、
多对抗，E 的有界性与收缩性均不可能被可塑性破坏**。安全由投影结构保证，与学习动力学
正交——为 A2 实现给出"先证后写"的契约：实现只需测试 Π 的正确性，无需重证系统安全。

*证明.* 检查定理 1–4 证明对 G 的全部引用：仅 (A3) 之界。Π 逐步强制该界。∎

**注 6.1（收敛主张的诚实界）.** 常步长投影随机逼近在有界噪声下**跟踪**慢变最优点至
O(α) 邻域（标准结果）；聊天场景无平稳性，故本设计**只主张稳定 + 有界跟踪，不主张
收敛到最优**。时标排序要求：k̲·(回合间隔) ≫ α ≫ 反思漂移对 G(T) 的影响——学习快于
人格漂移、慢于情绪收缩，三层时标不倒挂（标定约束，入 conformance）。

**注 6.2（资格迹的必要性）.** 若 φ 恒 1，quality 的全局标量误差会同步推动全部 8 维增益
（credit assignment 失败，学到的只是全局兴奋度）。φ = 近期该维 |a| 活动 ⟹ 只有参与了
本轮情绪反应的维度领赏罚。这是把"学习"从噪声放大器变成信用分配器的结构件。

## 7. 迟滞标签：换标频率界

标签解析（`resolve_label`）：键维量化 + Schmitt 死区 θ_h——想换档必须越过量化边界
≥ θ_h。

**命题 7（抗抖界）.** 同一键维两次**反向**换档之间，u 必须跨越宽度 2θ_h 的带。设单回合
位移上界 s̄（饱和步 ≤ Gᵢ·|aᵢ| ≤ 1 与衰减步 ≤ (1−β)·|u−ū| 之和的上确界，标定后典型
s̄ ≪ 1），则反向换档间隔 ≥ ⌈2θ_h/s̄⌉ 回合。θ_h = 0.08、s̄ ≈ 0.1 ⟹ 间隔 ≥ 2 回合：
边界噪声不可能逐回合翻转标签。

**注 7.1.** 单次大事件（|a| 大）仍可一步换标——这是特性不是缺陷：真实事件应当立即
反映；死区只滤边界徘徊。

## 9. 定位与具名贡献

对标：WASABI（Becker-Asano 2008，PAD 空间心境向人格基线指数衰减）、ALMA（Gebhard 2005，
人格推导的心境均衡 + 低通）、EMA/OCC/FAtiMA（appraisal 谱系）、Park et al. 2023
（poignancy/reflection，作用于记忆重要度与自由文本反思，非数值特质 ODE）。

**具名贡献（本标准 claim 的、且仅 claim 的三件）：**

- **C1 scar 耦合时间常数调制**：情绪衰减半衰期由不可逆伤痕账本调制、封顶、自愈
  （定理 3）。四镜头评审确认在上列具名系统中无对应物。
- **C2 锚回弹有界人格可塑性**：漂移半径定理（定理 5）给出"可被生活弯折、不可被
  改写"的闭式安全界——区别于 Park 的记忆反思（无数值界）与 ALMA 的静态人格。
- **C3 分离定理**：情绪有界性与增益学习解耦（定理 6）——允许在不重证系统安全的
  前提下引入在线可塑性。

**命名撞车处置**：poignancy/reflection 沿用 Park 词汇但语义不同（数值特质 ODE vs
记忆反思树）；标准文本中首次出现处加差异化脚注，中文名采"沉淀量/回望"以避免
false equivalence。其余元素（衰减、饱和、circumplex、迟滞、凸传染）**明示为经典机制
的工程实例**，不做新颖性主张——诚实定位是本仓库的既定纪律。
